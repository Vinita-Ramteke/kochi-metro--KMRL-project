  🚆 Kochi Metro – Document Automation System

A prototype web application built to automate document processing across departments with real-time monitoring and AI-based insights.

 🔗 Live Demo
👉 https://vinita-ramteke.github.io/kochi-metro-KMRL-project/

  💻 GitHub Repository
👉 https://github.com/Vinita-Ramteke/kochi-metro-KMRL-project


  📌 Features

- 🔐 Secure Login System (Employee ID based)
- 📊 Real-time Dashboard Monitoring
- 🧠 AI-based Document Analysis & Summary
- 🏢 Department-wise Document Tracking
- 📑 Automated Reports Generation
- ⚙️ Settings & Data Source Management (Gmail, WhatsApp, Teams)

 📌 Project Objective

To simulate a smart document automation system for Kochi Metro that:
- Reduces manual document handling
- Improves efficiency across departments
- Uses AI for faster decision-making

 

 🚀 Future Enhancements

- Real API connections (Gmail, WhatsApp, Teams)

- Role-based authentication system
